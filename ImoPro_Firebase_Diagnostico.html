Regras Firestore
Configure as regras no Firebase Console
Se o banco retornar erro de permissão, acesse:
Firebase Console → Firestore → Rules
e use estas regras para desenvolvimento: 
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /{document=**} {
      allow read, write: if true;
    }
  }
}
⚠️ Para produção, use autenticação!
